          <li>لبطانة السقف: تأكد من فك اللوح وإزالة الفوم القديم.</li>
          <li>للمقاعد: تأكد من عمل الفوم/الدعامة مقابل تغيير الغطاء فقط.</li>
        </ul>
      </details>

      <details class="details-card">
        <summary>
          <span>كيف يفشل العمل الرخيص عادةً</span>
          <span class="details-card__summary-text">التحضير والبنية والملاءمة هي ما يتجاهله العمل الرخيص.</span>
        </summary>
        <ul>
          <li>الغراء يفشل لأن السطح لم يُنظف ولم يُحضّر جيدًا.</li>
          <li>الخياطة تفشل لأن مناطق الشد لم تُدعّم أو لأن الملاءمة غير صحيحة.</li>
          <li>تظهر تموجات وتجاعيد لأن الشد والتركيب تم بسرعة.</li>
          <li>لا تتحسن الراحة لأن الفوم لم يُعالج.</li>
          <li>يعود نفس التشقق لأن الفوم/الشد تحت السطح بقي كما هو.</li>
        </ul>
        <p>الرخيص لا يعني دائمًا سيئًا، لكن العمل الذي يدوم يحتاج تحضيرًا وبنيةً وملاءمةً وطريقة واضحة.</p>
      </details>
    </div>
  </details>

  <details id="ref-process" class="issue-card">
    <summary class="issue-card__summary">
      <span class="issue-card__summary-title">الخطوات وما تتوقعه</span>
      <span class="issue-card__summary-text">ما الذي ترسله، كيف تكون المدة، وما الذي تراجعه عند الاستلام.</span>
    </summary>
    <div class="issue-card__body">
      <details class="details-card">
        <summary>
          <span>كيف تطلب تسعيرًا يسهل الرد عليه</span>
          <span class="details-card__summary-text">سنة/نوع/موديل + 3 صور + جملة واحدة واضحة.</span>
        </summary>
        <p>اجعلها قصيرة وواضحة ومدعومة بالصور. لا تحتاج مصطلحات تقنية.</p>
        <p>أرسل:</p>
        <ul>
          <li>سنة / نوع / موديل</li>
          <li>لقطة عامة (المقعد كاملًا / السقف كاملًا / لوحة الباب كاملة)</li>
          <li>لقطة قريبة (نمط التلف)</li>
          <li>صورة جانبية (ارتفاع المقعد / الترهل / الاعوجاج)</li>
          <li>جملة واحدة تصف المشكلة</li>
        </ul>
        <p>أمثلة:</p>
        <ul>
          <li>“قماش السقف مترهل في معظم المنطقة.”</li>
          <li>“مقعد السائق مسطح وأجلس أقل من قبل.”</li>
          <li>“حافة المقعد الخارجية متشققة والفوم هناك لين.”</li>
          <li>“لوحة الباب تتقشر من الأعلى وتبدو معوجة.”</li>
        </ul>
      </details>

      <details class="details-card">
        <summary>
          <span>المواعيد والمدة</span>
          <span class="details-card__summary-text">حجم العمل والخامة يحددان الجدول.</span>
        </summary>
        <p>المدة تعتمد على حجم العمل وتوفر الخامة.</p>
        <ul>
          <li>بطانة السقف غالبًا تكون سريعة بعد توفر الخامة.</li>
          <li>إعادة بناء الفوم تحتاج وقتًا أطول بسبب الفك والتشكيل وتجارب الملاءمة.</li>
          <li>تجديد داخلية كاملة غالبًا يحتاج تخطيطًا على مراحل.</li>
        </ul>
        <p>العمل السطحي أسرع؛ إعادة البناء أبطأ لأنها تعيد البنية لا الغطاء فقط.</p>
      </details>

      <details class="details-card">
        <summary>
          <span>كيف يبدو العمل الجيد عند الاستلام</span>
          <span class="details-card__summary-text">الملاءمة والخياطة والإحساس يجب أن تكون نظيفة ومتناسقة.</span>
        </summary>
        <ul>
          <li>خياطة مستقيمة ومتناسقة</li>
          <li>ألواح مشدودة بدون فقاعات أو تجاعيد عشوائية</li>
          <li>حواف نظيفة بدون آثار غراء أو قص سيئ</li>
          <li>تركيب القطع الداخلية بشكل محكم (بدون فراغات أو أصوات جديدة)</li>
          <li>إحساس المقعد متوازن (عاد الدعم، لا الشكل فقط)</li>
        </ul>
        <p>إذا بدا شيء غير صحيح فورًا، فمن الأسهل إصلاحه في نفس الوقت بدلًا من بعد أسابيع.</p>
      </details>

      <details class="details-card">
        <summary>
          <span>العناية بعد العمل وطول العمر</span>
          <span class="details-card__summary-text">العناية الأساسية تبطئ الأعطال المتوقعة.</span>
        </summary>
        <ul>
          <li>قد يبدو التنجيد جديدًا أكثر صلابة في البداية، خصوصًا بعد إعادة بناء الفوم.</li>
          <li>“استقرار” بسيط طبيعي؛ تغير كبير غير طبيعي.</li>
          <li>التنظيف المنتظم يمنع الزيوت والأوساخ من تسريع الجفاف والاهتراء.</li>
          <li>الحرارة لفترات طويلة تسرّع شيخوخة الغراء والفوم؛ الظل يساعد.</li>
        </ul>
        <p>طول العمر يعتمد على الطريقة والظروف: بنية جيدة + خامة مناسبة تعيش أكثر.</p>
      </details>
    </div>
  </details>
</div>

  const openDetailsFromHash = () => {
    const hash = window.location.hash;
    if (!hash) return;

    const target = document.querySelector(hash);
    if (!target) return;

    const details =
      target.tagName === 'DETAILS' ? target : target.closest('details');
    if (!details || details.tagName !== 'DETAILS') return;

    details.open = true;
  };

  window.addEventListener('DOMContentLoaded', openDetailsFromHash);
  window.addEventListener('hashchange', openDetailsFromHash);
})();
